# Image-Gallery

An Amazing Image Gallery where you can upload images, delete and Update Images..

## Features

- Live previews
- Fullscreen mode
- Cross platform
- Completely Responsive
- Images will be saved in database
- Generate Cloudinary Image link

## Technologies

- Tailwindcss
- React.js
- Prisma
- PostgresQL
- Cloudinary
- Multer

## Installation

Install Image-Gallery with npm

# Front-End

```bash
  clone this repo
  cd Image-Gallery
  cd client
  npm install
  npm start
```
# Back-End

```bash
  clone this repo
  cd Image-Gallery
  cd server
  npm install
  npm run dev
```
